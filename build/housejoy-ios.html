Booking a Plumber

Booking a plumber was a really simple workflow but we wanted to use the workflow to give users a sense of the cost by productising the service and also capture information that will help assigning the right technician. We made sure that they are prepared about the time and materials they need for the job.

Booking Beauty Service

Beauty was one of the complicated workflows as we wanted to let the users select multiple services, see details about each, select quantities etc. without complicating it and making it a shopping cart.

After multiple iterations and explorations, we came to this accordion based multi-select interaction which is fast and fun to use.

Feedback flow with skip logic

We wanted to make feedback collection smart and not a standard 5 star rating with a text box. Hence, we created a flow of skip logic based questions which change based on the answer by the user to capture really meaningful feedback.
